# Task 1 — Onboarding and Environment Setup

> **ApexPlanet Software Pvt. Ltd. | Android App Development Internship**
> **Timeline:** Days 1–9 (9 Days)

---

## 📋 Description

This first task covers your onboarding into the internship program and the complete setup of your Android development environment. By the end of these 9 days, you should be comfortable with Android Studio, have a working development environment, and understand the basics of Android app architecture.

---

## 🎯 Objectives

- Complete company onboarding and understand internship goals
- Install and configure all tools required for Android development
- Learn Android development fundamentals (activities, fragments, layouts)
- Initialize your GitHub repository for the internship project

---

## 📅 Day-by-Day Breakdown

### Days 1–2: Onboarding
- Attend orientation sessions to understand company culture and internship expectations
- Get introduced to your team members and assigned mentor
- Review the internship roadmap and deliverable schedule

### Days 3–5: Environment Setup
- Install **Android Studio** (latest stable version)
- Install required SDKs and tools:
  - Java Development Kit (JDK) or Kotlin plugin
  - Gradle build system
  - Android Emulator / AVD Manager
- Install and configure **Git**
- Create a **GitHub** account (if you don't already have one)
- Familiarize yourself with the Android Studio interface: Project Explorer, Editor, Logcat, and Emulator

### Days 6–9: Basics of Android Development
- Study core Android concepts:
  - **Activities** — the building blocks of an Android app's UI
  - **Fragments** — modular UI sections within an Activity
  - **Layouts** — XML-based UI structure (LinearLayout, ConstraintLayout, etc.)
- Complete at least one introductory tutorial (e.g., Google's "Build Your First App")
- Build one or two small practice projects in Android Studio

---

## ✅ Prerequisites

- A laptop/desktop with at least **8 GB RAM** (16 GB recommended)
- **8+ GB free disk space** for Android Studio and SDKs
- Stable internet connection for SDK downloads
- Basic understanding of Java or Kotlin (helpful but not mandatory)
- A GitHub account — [Sign up here](https://github.com)

---

## 🛠️ Installation Steps

1. **Download Android Studio** from [developer.android.com/studio](https://developer.android.com/studio)
2. Run the installer and follow the setup wizard — ensure you include the Android Virtual Device (AVD)
3. Open Android Studio → **SDK Manager** → install the latest Android SDK platform
4. Install **Git** from [git-scm.com](https://git-scm.com)
5. Configure Git with your identity:
   ```bash
   git config --global user.name "Your Name"
   git config --global user.email "you@example.com"
   ```
6. Create a new GitHub repository named `apexplanet-android-internship` and set it to **Public**
7. Clone it locally:
   ```bash
   git clone https://github.com/<your-username>/apexplanet-android-internship.git
   ```
